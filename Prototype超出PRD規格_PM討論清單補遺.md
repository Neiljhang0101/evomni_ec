## 摘要

| 模組 | 功能 | 建議處置 |
|---|---|---|
| 儀表板 | 快速導覽入口格（Quick Navigation Grid） | 補回儀表板 PRD |
| 儀表板 | 低庫存警示列表直接呈現於 Dashboard | 補回儀表板 PRD |
| 商品中心 — 商品管理 | KPI 卡片整合於産品中心首頁 | 補回商品中心 PRD 或形成決策記錄（移除） |

| 行銷中心 — 一頁式商店 | Landing Page 詳情頁（lp-detail-v2） | 補回一頁式商店 PRD 或形成決策記錄（整合至列表 Drawer） |
