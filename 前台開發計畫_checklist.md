## 一、會員認證流程

> 相關 PRD：[前台會員登入 PRD](PRD_new/Evomni_前台會員登入_PRD.md)（v1.3）、[Part6 會員管理 PRD §6.9](PRD_new/Evomni_Part6_會員管理_PRD.md)（註冊流程）

### 登入頁 `/login`

- [ ] Email + 密碼登入表單
- [ ] 第三方快速登入按鈕（LINE / Google / Facebook）
- [ ] 「忘記密碼？」連結
- [ ] 「立即註冊」連結
- [ ] `redirect_url` 機制（登入後返回原頁，如結帳頁）
- [ ] 登入成功後自動合併訪客購物車至會員購物車
- [ ] 錯誤態：帳號或密碼錯誤
- [ ] 錯誤態：帳號尚未驗證（Email 模式 / 人工審核模式文案不同）
- [ ] 錯誤態：帳號暫鎖 15 分鐘（含倒數計時顯示）
- [ ] 錯誤態：帳號系統自動停用
- [ ] 錯誤態：速率限制（IP 層，10 分鐘冷卻）
- [ ] CAPTCHA 觸發（Google reCAPTCHA v2，錯誤 N 次後出現）

### 忘記密碼頁 `/forgot-password`

- [ ] Email 輸入框 + 「發送重設密碼信」按鈕
- [ ] 「返回登入」連結
- [ ] 送出後同頁切換：發送成功提示 + 60 秒倒數「重新發送」按鈕

### 密碼重設頁 `/reset-password?token=xxx`

- [ ] 新密碼輸入框（含顯示/隱藏密碼切換）
- [ ] 密碼強度指示條（弱 / 普通 / 強 / 非常強）
- [ ] 確認新密碼輸入框
- [ ] Token 過期態：過期警告 + 「重新申請」按鈕
- [ ] 更新成功態：確認頁 + 「前往登入」按鈕

### 帳號驗證頁 `/verify-email?token=xxx`

- [ ] Token 有效：啟用成功頁 + 「前往登入」按鈕
- [ ] Token 無效 / 已使用：失效提示 + 「重發驗證信」按鈕

### 重發驗證信頁 `/resend-verification`

- [ ] Email 輸入框（從登入頁帶入時自動填入）
- [ ] 送出後：發送成功提示 + 60 秒倒數「重新發送」按鈕
- [ ] 「返回登入」連結

## 八、共用狀態規範

| 狀態 | 使用情境 | 視覺表現 |
|------|---------|--------|
| 成功 | 登入成功 / 儲存成功 / 加入購物車成功 | 綠色 Toast |
| 錯誤 | 登入失敗 / 庫存不足 / 表單驗證錯誤 | 紅色 Alert / Toast |
| 警告 | 低庫存 / 帳號暫鎖 / 密碼強度弱 | 橙色 Tag / Alert |
| 提示 | 即將到期 / 僅剩 N 件 / 需登入才能使用 | 藍色 Tooltip / Alert |
| 載入中 | 上傳中 / 查詢中 / 登入中 | Spinner / Progress Bar |
| 空態 | 購物車為空 / 無訂單 / 無優惠券 | 插圖 + 說明文字 + 行動按鈕 |

## 九、跨頁面主要流程

| 流程 | 經過頁面 |
|------|---------|
| 新用戶購買 | 商品列表 → 商品詳情 → 購物車 → 登入 → 結帳 → 訂單確認 |
| 既有會員購買 | 商品列表 → 商品詳情 → 購物車 → 結帳 → 訂單確認 |
| 訪客購買 | 商品列表 → 商品詳情 → 購物車 → 結帳（填入資訊）→ 訂單確認 |
| 忘記密碼 | 登入頁 → 忘記密碼 → Email 驗證 → 密碼重設 → 登入頁 |
| 帳號驗證 | 收驗證信 → `/verify-email` → 成功頁 → 登入頁 |
| 社群登入（有合併需求）| 登入頁 → OAuth → 帳號合併確認 → 密碼驗證 → 原頁 |
| 退換貨申請 | 會員中心訂單詳情 → 申請表單 → 確認送出 → 等待審核狀態 |
| 刪除帳號 | 帳號安全 → 申請確認 → 收確認信 → 點擊連結 → 30 天冷靜期 |
